## Description

Una REST API creada con NestJS para el Laboratorio de [OpenWebinars](https://openwebinars.net): **Crea una REST API con NestJS, Postgres, MongoDB y Docker**

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
